# vi basic
▪ 0(^)	: 라인의 처음 문자(라인의 처음)으로 이동
▪ $	: 라인의 마지막으로 이동

▪ H      : 현재 페이지의 가장 첫번째 줄로 이동
▪ L      : 현재 페이지의 가장 마지막 줄로 이동

▪ 1G	: 문서의 첫 번째 라인으로 이동
▪ G	: 문서의 마지막 라인으로 이동
